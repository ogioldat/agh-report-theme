
# AGH Report Theme

A for of  @mzsuetam's pandoc-agh-report-theme.

## Personalized improvements

- [x] Basic Docker setup.
- [x] VSCode tasks.
- [ ] Don't require rebuild for specific project.

## Usage

1. Build the container `docker build -t report-doc-builder .`
2. Add `tasks.json` to VSCode.
3. Render lab using shortcut `Cmd` + `Shift` + `B`.
